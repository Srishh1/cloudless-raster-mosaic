# Cloudless Raster Mosaic Generation

This project generates a cloud-free satellite mosaic by processing and combining multiple raster images.

The system removes cloud-covered pixels and produces a clean composite image using geospatial data processing techniques.

---

## 🚀 Project Overview

- Processed multi-temporal satellite raster images
- Applied cloud masking techniques
- Generated cloud-free mosaic composite
- Optimized raster stacking and pixel selection
- Handled large geospatial datasets efficiently

---

## 🛠 Tech Stack

- Python
- NumPy
- Rasterio / GDAL
- Matplotlib
- Jupyter Notebook
